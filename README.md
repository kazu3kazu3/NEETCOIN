NEETCOIN is a PoS/POW hybrid cryptocurrency.
## SPEC ##

+ Algorithm: Scrypt

+ POS annual interest: 21%

+ Superblock 210%(POS)　　<br>
　10000 ~ 12100　　　　<br>
　20000 ~ 22100　　　　　<br>
　40000 ~ 42100　　　　　<br>
　70000 ~ 72100　　　　<br>
　100000 ~ 102100　  <br>
  After this, 2100 super blocks every 50000 blocks

+ RPCPort: 21010

+ Port: 21011

## How to run tests

```bash
mkdir build && cd build
cmake ..
make
./test_neetcoin --log_level=message
```
