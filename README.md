# NEETCOIN

NEETCOIN is a PoS/POW hybrid cryptocurrency.

## SPEC

- Algorithm: Scrypt
- POS annual interest: 21%
- Superblock 210%(POS)
  - 10000 ~ 12100
  - 20000 ~ 22100
  - 40000 ~ 42100
  - 70000 ~ 72100
  - 100000 ~ 102100
  - After this, 2100 super blocks every 50000 blocks
- RPCPort: 21010
- Port: 21011
